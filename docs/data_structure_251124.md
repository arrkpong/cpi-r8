# Data Dictionary: 251124_R8_FDN_Load_CG_Calc_V01.csv

ไฟล์นี้เป็น **ไฟล์ข้อมูลหลัก (Base Data)** ที่เก็บข้อมูลตำแหน่งและน้ำหนักบรรทุก (Load) ของฐานรากและเสาตอม่อ

## โครงสร้างข้อมูล (Columns)

| ชื่อคอลัมน์ (Header) | ความหมาย (Description)                | หน่วย (Unit) | ตัวอย่างข้อมูล     | คำอธิบายเพิ่มเติม                                 |
| :------------------- | :------------------------------------ | :----------- | :----------------- | :------------------------------------------------ |
| **SN**               | **ลำดับที่** (Serial Number)          | -            | `1`, `2`           | รหัสลำดับข้อมูล (ใช้เป็น ID อ้างอิงได้)           |
| **Bld**              | **โซนอาคาร** (Building/Zone)          | -            | `A`, `B`, `C`      | ระบุโซนของอาคาร                                   |
| **FD**               | **รหัสฐานราก** (Foundation ID)        | -            | `A_FG03`           | ชื่อเรียกฐานราก                                   |
| **VC**               | **รหัสเสา/ผนัง** (Vertical Component) | -            | `A_C19`, `A_SW1P3` | ชื่อเรียกเสาหรือผนัง (ใช้จับคู่กับไฟล์ Dimension) |
| **Xvc**              | **พิกัด X**                           | เมตร (m)     | `135.45`           | ตำแหน่งในแกน X                                    |
| **Yvc**              | **พิกัด Y**                           | เมตร (m)     | `16.472`           | ตำแหน่งในแกน Y                                    |
| **Pvc**              | **น้ำหนักบรรทุก** (Load)              | ตัน (Ton)    | `-139.85`          | น้ำหนักที่กระทำลงสู่ฐานราก (ค่าลบ = แรงอัด)       |

## ข้อสังเกต (Notes)

1.  **ความครอบคลุม (Coverage):**

    - ไฟล์นี้มีข้อมูลตำแหน่งของ **ทุกองค์ประกอบ** (เสา C, ผนังรับแรงเฉือน SW, ผนังทั่วไป CW)
    - ใช้เป็นแหล่งข้อมูลหลักในการระบุตำแหน่ง (Location Source)

2.  **ความสัมพันธ์ (Relation):**
    - คอลัมน์ `VC` ใช้เชื่อมโยงกับคอลัมน์ `Pier` ในไฟล์ `251125_R8_ABCD_VC_Location_Dimension_V01.csv` เพื่อดึงขนาดจริงมาแสดงผล
