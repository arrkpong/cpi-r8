# Data Dictionary: 251125_R8_ABCD_VC_Location_Dimension_V01.csv

ไฟล์นี้เก็บข้อมูล **ขนาด (Dimension)** และ **ตำแหน่ง (Location)** ของโครงสร้างเสา (Column) และผนังรับแรงเฉือน (Shear Wall)

## โครงสร้างข้อมูล (Columns)

| ชื่อคอลัมน์ (Header) | ความหมาย (Description) | หน่วย (Unit)  | ตัวอย่างข้อมูล     | คำอธิบายเพิ่มเติม                                     |
| :------------------- | :--------------------- | :------------ | :----------------- | :---------------------------------------------------- |
| **Pier**             | **รหัสระบุตำแหน่ง**    | -             | `C_C01`, `A_SW1P3` | ใช้สำหรับจับคู่ (Join) กับไฟล์ข้อมูลหลัก (Load Data)  |
| **AxisAngle**        | **มุมหมุน**            | องศา (Degree) | `90.00`, `0.00`    | มุมการวางตัวของโครงสร้างเทียบกับแกน X                 |
| **Len.**             | **ความยาว**            | เมตร (m)      | `0.40`, `19.65`    | ขนาดความกว้างของหน้าตัด (ใช้เป็น Width ในการวาด)      |
| **Thick.**           | **ความหนา**            | เมตร (m)      | `0.35`, `0.11`     | ขนาดความลึกของหน้าตัด (ใช้เป็น Depth/Height ในการวาด) |
| **CG_X**             | **พิกัด X**            | เมตร (m)      | `-120.75`          | ตำแหน่งจุดศูนย์ถ่วงในแกน X                            |
| **CG_Y**             | **พิกัด Y**            | เมตร (m)      | `13.35`            | ตำแหน่งจุดศูนย์ถ่วงในแกน Y                            |

## ข้อสังเกต (Notes)

1.  **ประเภทข้อมูล (Data Types):**
    - **Columns (เสา):** รหัสขึ้นต้นด้วย `C_` (เช่น `C_C01`)
    - **Shear Walls (ผนังรับแรงเฉือน):** รหัสขึ้นต้นด้วย `SW` (เช่น `A_SW1P3`)
2.  **การนำไปใช้ (Usage in Visualization):**

    - **Shape:** วาดเป็นสี่เหลี่ยมผืนผ้า (Rectangle)
    - **Size:** กว้าง = `Len.`, สูง = `Thick.`
    - **Rotation:** หมุนตามค่า `AxisAngle`
    - **Position:** วางที่จุด `(CG_X, CG_Y)`

3.  **ข้อมูลที่ขาดหาย (Missing Data):**
    - ไม่พบข้อมูลสำหรับผนังทั่วไป (General Walls) ที่รหัสขึ้นต้นด้วย `CW` ในไฟล์นี้
