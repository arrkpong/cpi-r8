# Data Dictionary: 251125_R8_ABCD_Grid Definitions - Grid Lines.csv

ไฟล์นี้เก็บข้อมูล **เส้นกริด (Grid Lines)** สำหรับแต่ละโซนอาคาร เพื่อใช้ในการอ้างอิงตำแหน่งและแสดงผลบนแปลน

## โครงสร้างข้อมูล (Columns)

| ชื่อคอลัมน์ (Header) | ความหมาย (Description)        | หน่วย (Unit) | ตัวอย่างข้อมูล                   | คำอธิบายเพิ่มเติม                                                        |
| :------------------- | :---------------------------- | :----------- | :------------------------------- | :----------------------------------------------------------------------- |
| **Name**             | **ชื่อโซน** (Zone Name)       | -            | `GA`, `GB`, `GC`                 | รหัสโซนของอาคาร (GA=Zone A, GB=Zone B, GC=Zone C)                        |
| **Grid Line Type**   | **ประเภทเส้นกริด**            | -            | `X (Cartesian)`, `Y (Cartesian)` | ระบุแกนของเส้นกริด (X = เส้นตั้ง, Y = เส้นนอน)                           |
| **ID**               | **ชื่อเส้นกริด** (Grid Label) | -            | `R8.A`, `1`, `A1`                | ชื่อเรียกของเส้นกริดนั้นๆ                                                |
| **Ordinate**         | **พิกัด** (Coordinate)        | เมตร (m)     | `0`, `4.05`, `11.8`              | ระยะตำแหน่งของเส้นกริด (เป็น Local Coordinate เริ่มต้นที่ 0 ของแต่ละโซน) |

## ข้อสังเกต (Notes)

1.  **พิกัด (Coordinates):**

    - ค่า `Ordinate` เป็น **Local Coordinate** (พิกัดท้องถิ่น) ของแต่ละโซน
    - หากต้องการนำไปวางซ้อนกับข้อมูลเสา/ผนัง (`floorData`) ที่เป็น **Global Coordinate** จำเป็นต้องบวกค่า **Offset** เข้าไป:
      - **Zone A (GA):** +10.0 m
      - **Zone B (GB):** -45.2 m
      - **Zone C (GC):** -120.8 m

2.  **การใช้งาน (Usage):**
    - ใช้สำหรับวาดเส้นอ้างอิง (Reference Lines) บนหน้าจอ
    - ข้อมูลถูกแปลงเป็น `window.gridData` ในไฟล์ `data.js` โดยสคริปต์ `tools/process_grid_data.py`
